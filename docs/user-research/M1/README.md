# M1 pilot research notes

One file per pilot landlord, named `<first-name>.md` (e.g. `sarah.md`), using
the template below. This is Craig's to run (M1-W4-03) — the app ships an
in-product **Feedback** button that emails `NEXT_PUBLIC_FEEDBACK_EMAIL`; those
emails get pasted into the relevant landlord's notes.
